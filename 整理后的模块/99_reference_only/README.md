# 99 仅参考资料

状态：`source-index`

## 模块目的

收纳许可证、来源、工程质量或题型特化程度暂不适合进入正式模块的候选。这里的内容可以用于理解思路、引脚经验和调参方法，但不能直接复制到正式代码。

## 归入规则

- 无许可证声明的仓库。
- GPL、CC BY-NC-SA 或其他与本仓库后续分发方式不确定的来源。
- 强绑定具体学校、板卡、比赛年份或硬件接线的工程。
- 含大量二进制、构建产物、闭源库或无法判断来源的文件。
- 代码风格可借鉴但需要完整重写的实现。
